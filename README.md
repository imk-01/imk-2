1. Install Node.js
2. Gunakan vscode, buka terminal, terminal cek:
   node -v
   npm -v
   **Pastikan keluar versi**
3. Install Dependencies
   npm install
4. Jalankan project
   npm run dev
5. Build dan Start (buka terminl baru dengan tidak menutup terminal sebelumnya)
   npm run build
   npm start
6. Buka browser dan akses localhost
